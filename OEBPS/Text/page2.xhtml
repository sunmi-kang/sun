알고리즘의 3가지 제어 구조

알고리즘을 설계할 때 사용하는 대표적인 제어 구조는 순차(Sequence), 선택(Selection), 반복(Iteration)입니다. 이 세 가지 구조를 조합하여 복잡한 문제도 논리적으로 해결할 수 있습니다.

1. 순차 구조 (Sequence)

명령문이 위에서 아래로 차례대로 실행되는 구조입니다.

print("Hello")
print("World")

2. 선택 구조 (Selection)

조건에 따라 실행 경로가 달라지는 구조입니다.

if score >= 60:
    print("합격")
else:
    print("불합격")

3. 반복 구조 (Iteration)

특정 조건이 참인 동안 명령문을 반복 실행하는 구조입니다.

for i in range(5):
    print(i)

이러한 제어 구조를 적절히 활용하면 다양한 문제를 효율적으로 해결할 수 있습니다.
